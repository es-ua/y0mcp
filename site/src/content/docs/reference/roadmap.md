## v1 — Local (current)

- Slack channel plugin for Claude Code
- Permission relay via Slack reactions
- Multi-project routing (one agent per channel)
- Always-on with launchd/systemd
- Auto token refresh with file locking
- Dozzle container log integration

## v2 — Cloud

- Hosted service at `mcp.y0mcp.dev`
- Connect agents to cloud infrastructure
- API key support alongside claude.ai subscriptions
- Web dashboard for managing agents
- Discord plugin (`y0discord`)
- Dozzle plugin (`y0dozzle`)
